# MetaParser

Parser for MetaLyceum

## Текущие источники

- Cyberleninka
- PhilPapers/PhilArchive
- JMPhil
- SSRN
- JSTOR

## Критерии отбора статей

- Указан автор и ключевые слова
- Присутствует abstract, в котором раскрыта тема статьи
- Кол-во листов не менее 10
- Опубликована не ранее 1980 года
- Публикация в авторитетном научном источнике/журнале
- Наличие метрик (просмотры, скачивания) и/или комментариев, цитирований
